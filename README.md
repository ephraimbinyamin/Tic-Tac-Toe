## Installation

To install the project on your computer, follow these steps:

1. Clone the repository:
```
git clone https://github.com/ephraimbinyamin/Tic-Tac-Toe.git
```

2. Go to project folder:
```
cd Tic-Tac-Toe
```

3. Install dependencies:
```
npm install
```

4. Start the project:
```
npm start
```
